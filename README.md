# Laptop Hosting Landing (Next.js + Tailwind)

Single-page landing site built with Next.js (App Router) and Tailwind CSS.

## Run
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Notes
- This is a one-page site with section anchors (About / Opportunity / Compensation / FAQ / Contact).
- Replace `public/hero.svg` with your real hero image if you have one.
